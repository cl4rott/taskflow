# TaskFlow

Aplicativo de gerenciamento de tarefas desenvolvido com Ionic.

## Funcionalidades

- Adicionar tarefas 
- Visualizar tarefas
- Filtrar tarefas
- Marcar como concluída

## Tecnologias

- Ionic
- Angular
- TypeScript

## Como executar

npm install  
ionic serve
